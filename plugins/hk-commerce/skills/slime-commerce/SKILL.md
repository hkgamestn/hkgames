## §2 · Le cadre durable (vrai quel que soit l'intake)

**Positionnement** — le slime est un produit SENSORIEL : la texture EST le produit, la vidéo EST
la preuve. La marque se joue au toucher filmé (pokes, stretchs, sizzle ASMR), pas au discours.
Différenciation crédible : textures signature nommées, drops limités, kits DIY, personnalisation.

**Confiance parents (l'acheteur n'est pas l'utilisateur)** — page dédiée sécurité : ingrédients
listés, sans-borax si vrai (jamais d'allégation non prouvée), âge conseillé, entretien/durée de
vie du slime, contact réactif. C'est LE déverrouilleur d'achat.

**Page produit (anatomie non négociable)** — vidéo texture en premier média · nom évocateur +
famille de texture (butter/cloud/clear/crunchy) · parfum · taille en ml/g · prix DT visible ·
badge stock/drop · avis avec photos/vidéos clients · cross-sell kit/activateur. Zéro template
générique : la DA suit anti-generic-standard (marque sensorielle ≠ boutique Shopify par défaut).

**Offres** — l'AOV se construit : bundles à thème, box mensuelle/mystère, kit anniversaire,
recharge activateur (consommable = rétention), carte cadeau. Rareté honnête (vrais drops datés).

**COD & logistique (réalité TN)** — le COD domine : confirmer chaque commande (appel/WhatsApp)
avant expédition pour tuer le taux de retour ; suivre le taux de retour par wilaya ; emballage
anti-chaleur l'été (un slime fondu = un avis assassin) ; politique SAV claire.

**Moteur de contenu** — le contenu organique est l'actif n°1 d'une marque slime :
cadence hebdo (restock/drop, ASMR pur, coulisses fabrication, UGC clients reposté, éducation
entretien) ; hooks en darija naturelle ; le catalogue de vidéos nourrit ensuite meta-ads-engine.

**Rétention** — fichier clients dès J1 (même un Sheet) : relance restock texture préférée,
anniversaires, statut VIP early-access sur les drops.
